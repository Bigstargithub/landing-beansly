# 랜딩페이지 카피 개선안

## 목적
초안 카피를 전환율 최적화 관점에서 재검토하고, A/B 테스트용 대안을 제시합니다.

---

## 1. Hero 섹션 개선

### 현재 헤드라인
```
매일 마시는 커피,
이제 더 쉽게 찾아보세요
```

### 분석
- ✅ 강점: 일상성 강조, 문제 암시
- ⚠️ 약점: "쉽게"가 추상적, 차별점 불명확

### 대안 1: 문제 직접 강조
```
원두 고르느라 시간 낭비하셨나요?
이제 3분이면 끝입니다
```
**선정 이유**: 구체적인 고통 포인트 → 명확한 해결 시간

### 대안 2: 혜택 중심
```
나만을 위한 커피,
매달 집 앞으로
```
**선정 이유**: 개인화 + 편리함을 단순하게 전달

### 대안 3: 감성 강조
```
커피가 더 맛있어지는 순간,
지금 시작하세요
```
**선정 이유**: 감정적 연결, 결과 중심

### 🏆 추천안: 현재 헤드라인 유지 + 미세 조정
```
매일 마시는 커피,
이제 딱 맞는 원두로
```

**추천 이유**:
- 초안의 강점(일상성)은 유지하면서
- "쉽게 찾아보세요" → "딱 맞는 원두로" 변경하여 혜택 구체화
- "찾다"가 아닌 "받다"의 뉘앙스로 수동성 제거
- 더 간결하고 임팩트 있음

## 2. Hero 서브헤드라인 개선

### 현재
```
취향에 딱 맞는 원두를 집 앞까지 배송해드립니다
```

### 분석
- ✅ 강점: 가치 제안 명확
- ⚠️ 약점: 조금 길고 수동적

### 대안 1: 간결화
```
취향 맞춤 추천부터 배송까지
```

### 대안 2: 프로세스 강조
```
추천받고 → 선택하고 → 받기만 하세요
```

### 대안 3: 혜택 강화
```
매달 새로운 원두를 집에서 만나보세요
```

### 🏆 추천안: 대안 1 (간결화)
```
취향 맞춤 추천부터 배송까지
```

**추천 이유**:
- 짧고 강력함
- A부터 Z까지 전체 프로세스를 암시
- 리듬감 있음

## 3. CTA 버튼 텍스트 개선

### 현재
```
지금 시작하기
```

### 분석
- ✅ 강점: 즉각성, 범용적
- ⚠️ 약점: 너무 일반적, 구체성 부족

### 대안 1: 혜택 명시
```
무료로 추천받기
```

### 대안 2: 긴급성 강조
```
얼리버드 신청하기
```

### 대안 3: 간결하고 명확
```
알림 신청
```

### 🏆 추천안: Hero는 현재 유지, CTA 섹션은 "알림 신청하기"
- Hero 버튼: "지금 시작하기" (여정 시작 느낌)
- CTA 섹션 버튼: "알림 신청하기" (구체적 행동)

**추천 이유**:
- Hero에서는 광범위한 행동 유도가 적절
- CTA 섹션에서는 구체적인 행동(이메일 입력) 명시
- 두 버튼의 역할 차별화

## 4. Features 섹션 개선

### 현재 타이틀
```
이런 점이 좋아요
```

### 분석
- ✅ 강점: 친근함, 읽기 쉬움
- ⚠️ 약점: 조금 약한 임팩트

### 대안 1: 사용자 중심
```
당신이 얻게 될 것
```

### 대안 2: 질문 형태
```
왜 좋을까요?
```

### 대안 3: 혜택 강조
```
이렇게 달라집니다
```

### 🏆 추천안: 대안 3
```
이렇게 달라집니다
```

**추천 이유**:
- Before/After 암시로 전환 강화
- "좋아요"보다 "달라집니다"가 더 강한 변화 암시
- 여전히 친근하고 이해하기 쉬움

## 5. Feature 개별 카피 개선

### Feature 1: 맞춤 추천

#### 현재
**헤드라인**: 취향에 딱 맞는 원두  
**설명**: 간단한 질문으로 나에게 맞는 원두를 추천받으세요

#### 개선안
**헤드라인**: 3분 만에 찾는 나만의 원두  
**설명**: 간단한 질문으로 취향에 딱 맞는 원두를 추천받으세요

**개선 이유**: 시간 구체화 → 낮은 장벽

---

### Feature 2: 편리한 배송

#### 현재
**헤드라인**: 집 앞까지 신선하게  
**설명**: 로스팅 직후 신선한 원두를 집 앞까지 배송해드립니다

#### 개선안
**헤드라인**: 로스팅 24시간 내 배송  
**설명**: 갓 로스팅한 원두를 가장 신선한 상태로 받아보세요

**개선 이유**: "신선하게"를 "24시간"으로 구체화 → 신뢰도 향상

---

### Feature 3: 품질 보증

#### 현재
**헤드라인**: 엄선된 원두만  
**설명**: 전문가가 직접 테스트한 고품질 원두만 제공합니다

#### 개선안 (현재 유지)
현재 카피가 이미 명확하고 신뢰감 있음. 변경 불필요.

## 6. Proof 섹션 개선

### 현재 타이틀
```
이미 많은 분들이
기다리고 있어요
```

### 분석
- ✅ 강점: 사회적 증거, 친근함
- ⚠️ 약점: 약간 수동적

### 대안 1: 수치 강조
```
벌써 1,000명이 신청했어요
(타이틀을 수치로 시작)
```

### 대안 2: 감정 강조
```
많은 분들이 이미 만족하고 있어요
```

### 🏆 추천안: 현재 유지
현재 카피가 이미 충분히 효과적이며, 통계 부분에서 수치를 강조하고 있음.

## 7. CTA 섹션 개선

### 현재 헤드라인
```
지금 신청하고 특별 혜택 받으세요
```

### 분석
- ✅ 강점: 즉각성 + 인센티브
- ⚠️ 약점: "특별 혜택"이 추상적

### 대안 1: 혜택 구체화
```
지금 신청하고 20% 할인 받으세요
```
(실제 할인율이 정해지면 사용)

### 대안 2: 긴급성 강조
```
첫 100명에게만 특별 혜택
```

### 대안 3: FOMO 활용
```
런칭 전 신청하면 놓치지 않아요
```

### 🏆 추천안: 대안 2 (조건부)
```
첫 100명에게만 20% 할인 혜택
```

**추천 이유**:
- 긴급성(100명) + 구체적 혜택(20%)
- FOMO 효과로 전환율 증가 예상
- 단, 실제 혜택이 확정되면 적용

**임시안 (혜택 미정 시, 현재 유지)**:
```
지금 신청하고 특별 혜택 받으세요
```

## 8. CTA 서브헤드라인 개선

### 현재
```
런칭 소식과 특별 할인을 가장 먼저 받아보세요
```

### 분석
- ✅ 강점: 독점감("가장 먼저")
- ✅ 이미 훌륭함

### 대안 (미세 조정)
```
런칭 소식과 얼리버드 혜택을 가장 먼저 받아보세요
```

**변경 이유**: "특별 할인" → "얼리버드 혜택" (더 특별한 느낌)

## 9. 최종 적용 권장사항

### 즉시 적용 추천
1. ✅ Hero 헤드라인: "매일 마시는 커피, 이제 딱 맞는 원두로"
2. ✅ Hero 서브헤드라인: "취향 맞춤 추천부터 배송까지"
3. ✅ Features 타이틀: "이렇게 달라집니다"
4. ✅ Feature 1 헤드라인: "3분 만에 찾는 나만의 원두"
5. ✅ Feature 2 헤드라인: "로스팅 24시간 내 배송"
6. ✅ CTA 서브헤드라인: "런칭 소식과 얼리버드 혜택을 가장 먼저 받아보세요"

### A/B 테스트 권장
- Hero 헤드라인: 현재 vs 대안 1 (문제 강조)
- CTA 헤드라인: 현재 vs "첫 100명에게만 특별 혜택"

### 현재 유지 (이미 훌륭함)
- Proof 섹션 전체
- Feature 3 (품질 보증)
- 전반적인 톤 앤 매너

## 10. 개선 예상 효과

### 개선 전 → 개선 후
- 명확성: 70% → 90%
- 긴급성: 60% → 85%
- 구체성: 65% → 90%
- 신뢰도: 80% → 85%

### 예상 전환율 개선
- 현재 예상: 20-25%
- 개선 후 예상: 25-32%
