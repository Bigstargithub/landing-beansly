# 랜딩페이지 카피 초안

## 작성 원칙
- **톤**: 간결하고 따뜻한 한국어
- **관점**: 사용자 입장에서 작성 (You, 당신이 아닌 "나", "우리")
- **강조**: 기술이 아닌 혜택과 감정
- **길이**: 한눈에 읽을 수 있는 짧은 문장

---

## 섹션별 카피

### 1. Hero 섹션

**헤드라인 (H1)**:
```
매일 마시는 커피,
이제 더 쉽게 찾아보세요
```

**서브헤드라인 (H2)**:
```
취향에 딱 맞는 원두를 집 앞까지 배송해드립니다
```

**CTA 버튼**:
```
지금 시작하기
```

**작성 의도**:
- "매일 마시는"으로 일상성 강조 → 공감대 형성
- "더 쉽게"로 문제 해결 암시 → 호기심 유발
- "취향에 딱 맞는"으로 개인화 강조 → 차별점 제시
- "집 앞까지"로 편리함 구체화 → 실용성 전달

---

### 2. Features/Value 섹션

**섹션 타이틀**:
```
이런 점이 좋아요
```

#### Feature 1: 맞춤 추천

**헤드라인**:
```
취향에 딱 맞는 원두
```

**설명**:
```
간단한 질문으로 나에게 맞는 원두를 추천받으세요
```

**작성 의도**:
- "취향에 딱"으로 개인화 경험 강조
- "간단한 질문"으로 참여 장벽 낮춤
- "나에게 맞는"으로 혜택을 자신에게 투영하도록 유도

---

#### Feature 2: 편리한 배송

**헤드라인**:
```
집 앞까지 신선하게
```

**설명**:
```
로스팅 직후 신선한 원두를 집 앞까지 배송해드립니다
```

**작성 의도**:
- "집 앞까지"로 편리함을 공간적으로 구체화
- "신선하게"로 품질 보증
- "로스팅 직후"로 전문성과 신선도 강조

---

#### Feature 3: 품질 보증

**헤드라인**:
```
엄선된 원두만
```

**설명**:
```
전문가가 직접 테스트한 고품질 원두만 제공합니다
```

**작성 의도**:
- "엄선된"으로 큐레이션 가치 전달
- "전문가가 직접"로 신뢰 구축
- "고품질만"으로 품질에 대한 확신 제공

---

### 3. Proof 섹션

**섹션 타이틀**:
```
이미 많은 분들이 기다리고 있어요
```

**사용자 후기 1**:
```
"정말 취향에 딱 맞는 원두를 받았어요"
- 김○○
```

**사용자 후기 2**:
```
"더 이상 원두 찾느라 헤매지 않아요"
- 이○○
```

**사용자 후기 3** (선택):
```
"매달 새로운 원두를 발견하는 재미가 있어요"
- 박○○
```

**대체안** (런칭 전):
```
벌써 1,000명 이상이 런칭을 기다리고 있습니다
```

**작성 의도**:
- 실제 사용자의 목소리로 진정성 확보
- 문제 해결 경험을 구체적으로 표현
- 사회적 증거로 신뢰도 향상

---

### 4. CTA 섹션

**헤드라인**:
```
지금 신청하고 특별 혜택 받으세요
```

**서브헤드라인**:
```
런칭 소식과 특별 할인을 가장 먼저 받아보세요
```

**입력 폼 플레이스홀더**:
```
이메일을 입력해주세요
```

**제출 버튼**:
```
알림 신청하기
```

**개인정보 안내**:
```
* 이메일은 런칭 알림 목적으로만 사용되며, 언제든 수신 거부할 수 있습니다
```

**제출 완료 메시지**:
```
신청 완료! 
곧 좋은 소식으로 찾아뵙겠습니다 ☕
```

**작성 의도**:
- "지금"으로 즉각적 행동 유도
- "특별 혜택"으로 인센티브 제공
- "가장 먼저"로 독점감 부여
- 개인정보 안내로 신뢰 확보
- 완료 메시지로 긍정적 마무리

```

```

```
나만의 완벽한 커피, 매달 집 앞으로
```

```

```
